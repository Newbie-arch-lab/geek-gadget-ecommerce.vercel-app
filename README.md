# GeeK-Gadget Website - Code Technologies Used

This premium electronics e-commerce website is built using the following technologies:

## Core Technologies

**HTML5** - The structural foundation of the website, providing semantic markup for all content sections including header, navigation, product grid, cart, and footer.

**CSS3** - Advanced styling with modern features including CSS Grid for responsive layouts, Flexbox for component alignment, CSS variables for consistent theming, gradient backgrounds, smooth transitions and animations, and backdrop filters for glassmorphism effects.

**JavaScript (ES6+)** - Powers all interactive functionality including dynamic product rendering, cart management with localStorage, search and filter systems, hero slider automation, modal windows, form validation, and real-time notifications.

## Design Framework

**Font Awesome 6.4.0** - Icon library providing scalable vector icons for navigation, products, social media, and UI elements throughout the site.

**Google Fonts (Outfit & Space Mono)** - Custom typography giving the site a modern, clean aesthetic with the Outfit font family for body text and headings.

## Key Features Implemented

**Responsive Grid System** - Auto-fitting product cards and category layouts that adapt to different screen sizes

**State Management** - Cart and wishlist data persisted using browser localStorage API

**Animation System** - Keyframe animations for slide-ins, fades, hover effects, and page transitions

**Dynamic Content Rendering** - JavaScript-powered product catalog with 7 featured items

**Interactive Components** - Quantity controls, product filters, search functionality, and wishlist toggling

**Modal System** - Sign-in forms and quick view product modals with backdrop blur effects

All code is vanilla JavaScript with no external frameworks, making it lightweight and fast-loading. The design follows modern web standards with mobile-first responsive breakpoints at 576px, 768px, and 1024px.
