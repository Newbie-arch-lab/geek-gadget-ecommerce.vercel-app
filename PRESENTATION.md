## üì± Responsive Design

The website is fully responsive with three main breakpoints:

- **Mobile**: < 576px (optimized for phones)
- **Tablet**: 576px - 1024px (optimized for tablets)
- **Desktop**: > 1024px (full desktop experience)

All components adapt seamlessly:
- Navigation menu transforms to hamburger menu on mobile
- Product grid adjusts column count based on screen size
- Typography scales appropriately
- Images and media are responsive

## üîß Technical Implementation Details

### Architecture
- **Single-Page Application (SPA)**: All functionality in one HTML file
- **No Build Process**: No compilation or bundling required
- **Zero Dependencies**: Pure vanilla JavaScript, HTML, and CSS
- **Lightweight**: Fast loading times with minimal external resources

### Key JavaScript Features
- **Product Data Structure**: JSON-like product objects with full details
- **Cart Management**: Add, remove, update quantities with localStorage persistence
- **Search Algorithm**: Real-time filtering of products by name
- **Category Filtering**: Dynamic product filtering by category
- **Hero Slider**: Automated carousel with manual navigation controls
- **Modal System**: Reusable modal component for various purposes
- **Event Delegation**: Efficient event handling

### State Management
- Uses browser `localStorage` API for data persistence
- Cart state persists across browser sessions
- Wishlist functionality with local storage
- No external state management libraries required
